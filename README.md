# Работа со строками на Rust

Этот проект представляет собой набор утилит для работы со строками на языке программирования Rust.

## Участники

- Михайленко Алексей (палиндром)

- Агапеева Дарья (перевод регистров)

- Резепов Петр (подсчет гласных)

## Использование

- Убедитесь, что у вас установлен компилятор _"Rust"_.

- Склонируйте проект

- Запустите, используя команду _`cargo run`_  
  
## Функции

Проект включает следующие функции:

- **Проверка на палиндром**: Функция анализирует строку и определяет, является ли она палиндромом.

- **Подсчет гласных**: Подсчитывает количество гласных букв в строке.
  
- **Перевод в верхний регистр**: Преобразует все символы строки в верхний регистр.

- **Перевод в нижний регистр**: Преобразует все символы строки в нижний регистр.
